Amizade


Qualquer Pessoa pode iniciar uma conversa com:

  • Homens
  • Mulheres
  • Ambos

A partir dos gostos de cada usuário o aplicativo indicará locais para que ambos possam se encontrar, com diversos beneficios

  • Homens Ativos
  • Mulheres Ativas

O Aplicativo indicará estabelecimentos como:

  • Restaurantes
  • Cafe
  • Show
  • Beber
  • Acadêmia
  • Comer

* Para os Homens Ativos, não existirá a opção de escolher mulheres, pois esse não é o objetivo do App.
